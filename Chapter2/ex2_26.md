# Exercise 2.26: Which of the following are legal? For those that are illegal, explain why.

**(a) const int buf;**     
Illegal. A const variable must be initialized upon declaration.      
**(b) int cnt = 0;**      
Legal.      
**(c) const int sz = cnt;**       
Legal.      
**(d) ++cnt; ++sz;**      
++cnt, legal.      
++sz, illegal. A const cannot be changed once initialized.
